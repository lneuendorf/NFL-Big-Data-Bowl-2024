## Setup
1. Build the conda environment with the necessary Python packages:
```
conda env create -f environment.yml
```

2. Activate the environment "bdb2024":
```
conda activate bdb2024
```

3. Intall the IPython kernel:
```
python -m ipykernel install --user --name=bdb2024
```

4. Run JupyterLab:
```
jupyter lab
```

5. Select the environment 'bdb2024' as the kernel on the JupyterLab server.
